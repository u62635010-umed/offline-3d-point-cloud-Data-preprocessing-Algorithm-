# Offline 3D Point Cloud Data Preprocessing Algorithm

## 📌 Project Description
This project focuses on preprocessing raw drone mapping data to generate clean and usable 3D point-cloud data for terrain inspection and analysis.

The algorithm performs multiple preprocessing steps such as noise filtering, coordinate alignment, and scaling to improve spatial accuracy of the point-cloud data.

## 🚀 Features
- Process raw LiDAR / drone mapping CSV data
- Remove noise from point-cloud
- Coordinate alignment
- Data scaling for better accuracy
- Generate processed 3D point data
- Ready for visualization / terrain inspection
